# 量子太空杀网页端体验指南

本指南将详细介绍如何在网页端直接体验量子太空杀游戏的相关功能。

## 网页端功能概述

量子太空杀的网页端基于Django框架开发，主要提供以下功能：

- 📖 **游戏攻略**：浏览、搜索、发布游戏攻略
- 👤 **用户系统**：注册、登录、个人信息管理
- 💬 **聊天室**：实时聊天、创建房间、多人交流
- 🔍 **搜索功能**：按标题、内容、作者搜索攻略
- ⚙️ **管理功能**：用户个人攻略管理

## 启动网页端步骤

### 1. 安装依赖

首先需要安装Django项目的依赖：

```bash
# 在项目根目录执行
cd quantumspacewar
pip install -r requirements.txt
```

### 2. 启动Django Web服务器

```bash
# 在quantumspacewar目录下执行
python manage.py runserver
```

服务器启动成功后，你将看到类似以下输出：

```
Watching for file changes with StatReloader
Performing system checks...

System check identified no issues (0 silenced).
October 26, 2024 - 15:00:00
Django version 3.x.x, using settings 'quantumspacewar.settings'
Starting development server at http://127.0.0.1:8000/
Quit the server with CTRL-BREAK.
```

### 3. 访问网页端

打开浏览器，访问以下地址：

```
http://127.0.0.1:8000/
```

## 主要功能页面

### 首页
- **地址**：`http://127.0.0.1:8000/`
- **功能**：展示所有游戏攻略列表，支持搜索和排序

### 注册页面
- **地址**：`http://127.0.0.1:8000/register/`
- **功能**：新用户注册账号

### 登录页面
- **地址**：`http://127.0.0.1:8000/login/`
- **功能**：用户登录系统

### 发布攻略
- **地址**：`http://127.0.0.1:8000/add_guide/`
- **功能**：登录后发布新的游戏攻略

### 攻略详情
- **地址**：`http://127.0.0.1:8000/guide/<id>/`（将`<id>`替换为攻略ID）
- **功能**：查看攻略详细内容

### 我的攻略
- **地址**：`http://127.0.0.1:8000/my_guides/`
- **功能**：登录后查看和管理自己发布的攻略

### 聊天室
- **地址**：`http://127.0.0.1:8000/chat/`
- **功能**：进入默认聊天室进行实时交流

## 功能使用说明

### 浏览攻略
1. 访问首页，即可看到所有攻略列表
2. 使用搜索框可以按标题、内容或作者搜索攻略
3. 点击攻略标题可以查看详细内容

### 用户注册与登录
1. 点击页面右上角的"注册"按钮创建账号
2. 使用注册的账号登录系统
3. 登录后可以发布攻略和使用聊天室功能

### 发布攻略
1. 登录后点击"发布攻略"按钮
2. 填写攻略标题和内容
3. 点击"发布"按钮即可发布攻略

### 使用聊天室
1. 登录后点击"聊天室"按钮进入默认聊天室
2. 输入消息内容并点击"发送"按钮
3. 可以点击"创建房间"按钮创建新的聊天室

## 注意事项

1. 网页端目前主要提供游戏攻略和聊天功能，完整的游戏体验可能需要结合移动端应用
2. 开发服务器仅用于测试，请勿在生产环境中使用
3. 如果需要关闭服务器，在终端中按`CTRL-BREAK`组合键
4. 如需修改服务器端口，可以使用`python manage.py runserver 8080`命令指定端口

## 常见问题

### Q: 服务器启动失败怎么办？
A: 检查是否已正确安装依赖，确保在`quantumspacewar`目录下执行命令，检查数据库配置是否正确。

### Q: 无法访问网页怎么办？
A: 确保服务器正在运行，检查访问地址是否正确，确保防火墙没有阻止访问。

### Q: 聊天室消息不实时更新怎么办？
A: 刷新页面或检查网络连接，确保浏览器支持JavaScript。
